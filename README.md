## Proyecto de servicios de Productos Clientes 

Apollo Server
MongoDB, Mongoose y GraphQL

## Getting Started

```bash
npm i
```

```bash
npm run dev
```

Open [http://localhost:4000](http://localhost:3000) with your browser to see the result.
